    tensorstore.TensorStore.domain : IndexDomain

    Domain of the array.

    Example

    >>> dataset = await ts.open(
...     {
...         'driver': 'n5',
...         'kvstore': {
...             'driver': 'memory'
...         }
...     },
...     dtype=ts.uint32,
...     domain=ts.IndexDomain(shape=[70, 80], labels=['x', 'y']),
...     create=True)
>>> dataset.domain
{ "x": [0, 70*), "y": [0, 80*) }

    The bounds of the domain reflect any transformations that have been applied:

    >>> dataset[30:50].domain
{ "x": [30, 50), "y": [0, 80*) }
